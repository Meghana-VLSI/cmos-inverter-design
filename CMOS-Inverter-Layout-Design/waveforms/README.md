# Waveforms

## Overview

This folder contains the simulation waveforms of the **CMOS Inverter** obtained during SPICE simulation. The waveforms verify the inverter's functionality by illustrating the relationship between the input and output signals.

## Files Included

| File | Description |
|------|-------------|
| `input_waveform.png` | Waveform showing the input voltage signal applied to the CMOS inverter. |
| `output_waveform.png` | Waveform showing the output voltage response of the CMOS inverter. |
| `input_output_waveform.png` | Combined waveform displaying both input and output signals, demonstrating the inverter's logical inversion behavior. |

## Software Used

- Magic VLSI Layout Editor
- SPICE Simulator (NGSPICE)
- SCMOS Technology

## Purpose

The waveform images are used to:

- Verify the functional operation of the CMOS inverter.
- Observe the relationship between the input and output signals.
- Confirm correct logical inversion.
- Analyze the switching characteristics of the inverter.
- Validate the design through simulation.

## Outcome

The simulation waveforms confirm the correct operation of the CMOS inverter. The output waveform is the logical complement of the input waveform, demonstrating successful inverter functionality and proper switching behavior.
