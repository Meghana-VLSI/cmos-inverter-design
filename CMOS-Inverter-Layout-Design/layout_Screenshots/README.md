# Layout Screenshots

## Overview

This folder contains screenshots of the CMOS inverter physical layout and the Design Rule Check (DRC) verification performed using **Magic VLSI Layout Editor** with **SCMOS Technology**.

These images provide visual confirmation of the completed layout and demonstrate that the design satisfies the required SCMOS design rules.

## Files Included

| File | Description |
|------|-------------|
| `layout_view.png` | Screenshot of the completed CMOS inverter layout showing the physical implementation, transistor placement, routing, and interconnections. |
| `drc_clean.png` | Screenshot showing the Design Rule Check (DRC) results. The absence of DRC errors confirms that the layout complies with SCMOS design rules. |

## Software Used

- Magic VLSI Layout Editor
- SCMOS Technology

## Purpose

The screenshots are included to:

- Illustrate the completed CMOS inverter layout.
- Verify the physical implementation of the design.
- Demonstrate successful Design Rule Check (DRC) verification.
- Provide documentation of the layout design process.

## Outcome

The layout screenshots confirm that the CMOS inverter has been successfully implemented in Magic VLSI and passes all Design Rule Checks, making the design suitable for layout extraction and further verification.
