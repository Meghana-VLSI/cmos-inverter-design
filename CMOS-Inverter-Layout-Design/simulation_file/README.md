# Simulation Files

## Overview

This folder contains the SPICE simulation file of the **CMOS Inverter** used for circuit simulation and functional verification. The simulation file is generated from the CMOS inverter design and is compatible with SPICE-based circuit simulators.

## Files Included

| File | Description |
|------|-------------|
| `cmos.spice` | SPICE netlist file used to simulate the CMOS inverter and analyze its electrical behavior. |

## Software Used

- Magic VLSI Layout Editor
- SPICE Simulator (e.g., NGSPICE)
- SCMOS Technology

## Purpose

The simulation file is used to:

- Verify the functionality of the CMOS inverter.
- Analyze the input and output voltage characteristics.
- Evaluate switching behavior.
- Validate the design before fabrication.
- Support post-layout simulation using the extracted netlist.

## Outcome

The SPICE simulation confirms the correct operation of the CMOS inverter by verifying its switching characteristics and logical inversion behavior. The results demonstrate that the design functions as expected and is suitable for further VLSI verification and implementation.
