# Documentation

## Overview

This folder contains the complete documentation for the **CMOS Inverter Layout Design using Magic VLSI with SCMOS Technology** project. The documentation explains the design methodology, schematic creation, layout implementation, simulation, verification, and design validation process.

It provides a detailed reference for understanding the CMOS inverter design flow, from circuit design to physical layout and simulation.

- Magic VLSI Layout Design

- Input and Output Waveforms
- Design Verification
- Results and Conclusion

## Software Used

- LTspice XVII
- Magic VLSI Layout Editor
- NGSPICE
- SCMOS Technology

## Purpose

The documentation serves as a comprehensive guide to the CMOS inverter design process. It explains the schematic design, physical layout creation, design rule verification, SPICE simulation, and waveform analysis, providing a complete understanding of the project.

## Files Included

- Project Report (PDF)

## Outcome

The documentation demonstrates the successful implementation and verification of the CMOS inverter. It confirms that the design follows SCMOS design rules, passes Design Rule Check (DRC), and exhibits correct inverter functionality through SPICE simulation.
