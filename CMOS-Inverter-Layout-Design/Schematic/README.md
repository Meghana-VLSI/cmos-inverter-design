# Schematic

## Overview

This folder contains the schematic diagram of the **CMOS Inverter** designed using **LTspice**. The schematic represents the logical circuit implementation of the CMOS inverter before the physical layout design.

## File Included

| File | Description |
|------|-------------|
| `schematic.png` | Schematic diagram of the CMOS inverter designed in LTspice. |

## Schematic Description

The schematic consists of:

- PMOS transistor connected to the supply voltage (VDD).
- NMOS transistor connected to Ground (GND).
- Common gate connection serving as the input.
- Common drain connection serving as the output.
- DC power supply and input voltage source.

## Software Used

- LTspice XVII

## Purpose

The schematic is used to verify the circuit design and functionality before implementing the physical layout in Magic VLSI using SCMOS technology.

## Outcome

The LTspice schematic provides the reference circuit for simulation and serves as the basis for the CMOS inverter layout implementation.
