# CMOS Inverter Layout Design Using Magic VLSI Tool

## 📌 Project Overview
This project demonstrates the design, implementation, and verification of a CMOS inverter using Magic VLSI Tool based on SCMOS technology. The workflow follows a complete VLSI physical design flow starting from layout creation to circuit simulation using NGSpice.

The CMOS inverter is the most fundamental digital logic gate and is widely used in all digital systems.

## 🧠 Project Description
A CMOS inverter consists of a PMOS and an NMOS transistor connected in a complementary configuration. When input is LOW, output is HIGH and vice versa.

This project includes:

- CMOS inverter layout design in Magic VLSI
- Placement of PMOS and NMOS transistors
- Metal routing and contact formation
- Design Rule Check (DRC) verification
- Extraction of SPICE netlist
- NGSpice simulation for waveform analysis
